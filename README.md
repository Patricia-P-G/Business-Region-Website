# 🌍 Business Region DE-DK Website - MMD 2nd Project

---

## 📖 Project Overview

This project was developed as part of the Multimedia Design programme at SEA - Syddansk Erhversakademi.

Business Region DE-DK is a responsive web platform designed to strengthen cross-border cooperation between businesses in Denmark and Germany.

The website provides information about partner organizations, networking opportunities, events, media content, and membership opportunities while following modern web development practices, accessibility standards, and SEO best practices.

## 🛠️ Tech Stack & Tools

- **Frontend:** Vue 3
- **Routing:** Vue Router
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Version Control:** Git & GitHub
- **Design:** Figma

---

## ✨ Key Features

- Responsive design for mobile, tablet, and desktop devices
- Accessibility-focused implementation using semantic HTML and ARIA attributes
- SEO-friendly page structure
- Dynamic company directory with search and sorting functionality
- Media section with category filtering
- Membership application form
- Reusable component architecture
